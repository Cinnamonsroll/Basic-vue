# Basic-vue

## How to run

Enter the frontend folder
```bash
cd frontend
```
Install all packages
```bash
npm install
```
Build for production
```bash
npm run build
```
Enter the backend folder
```bash
cd ../backend
```
Install all packages
```bash
npm install
```
Enter the starting directory
```bash
cd ../
```
Start the website
```bash
node .
```
